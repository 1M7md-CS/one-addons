HOW TO MERGE THIS INTO YOUR FABRIC MOD (1.21.1)
================================================

COPY FILES
----------
  java/your/package/mixin/MultiPlayerGameModeMixin.java
    -> src/main/java/<your_package>/mixin/

  resources/nmc.mixins.json
    -> src/main/resources/

EDIT fabric.mod.json
-------------------
  Add:  "mixins": ["nmc.mixins.json"],

  No entrypoint, no lang file, no keybinding needed.

WHAT IT DOES
------------
  Removes the 5-tick (0.25s) client-side cooldown between block breaks.
  This is purely client-side — the server does not enforce this delay.
  Safe on Hypixel and other servers. Does NOT change per-block break speed,
  reach, or any other mechanics.

WHY NOT MORE?
-------------
  Hypixel's anti-cheat (Watchdog) actively detects:
  - Mining speed multipliers (timing analysis)
  - Reach extensions
  - Airborne penalty removal (speed consistency)
  - Any packet manipulation

  The cooldown removal is undetectable because it only removes an
  artificial client-side delay that servers never validate.
